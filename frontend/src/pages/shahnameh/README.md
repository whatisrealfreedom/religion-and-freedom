# Shahnameh Sections

این پوشه شامل بخش‌های مختلف داستان‌های شاهنامه است. هر بخش یک صفحه جداگانه دارد که می‌تواند مستقل از داستان اصلی نمایش داده شود.

## ساختار

```
shahnameh/
├── sections/
│   ├── JamshidReign.tsx      # بخش پادشاهی جمشید (jamshid-1)
│   └── ...                    # بخش‌های دیگر در آینده
└── README.md                  # این فایل
```

## Routes

هر بخش یک route جداگانه دارد:
- `/shahnameh/jamshid/reign` - پادشاهی جمشید

## مزایا

1. **صفحات جداگانه**: هر بخش یک صفحه مستقل است
2. **URL مستقل**: هر بخش URL مخصوص خود را دارد
3. **Refresh Safe**: با refresh صفحه، همان بخش نمایش داده می‌شود
4. **سازماندهی بهتر**: فایل‌ها در پوشه‌های مرتب قرار دارند
5. **قابل نگهداری**: هر بخش را می‌توان مستقل ویرایش کرد

## افزودن بخش جدید

برای افزودن بخش جدید:

1. فایل جدید در `sections/` بسازید
2. Route را در `App.tsx` اضافه کنید
3. لینک را در `ShahnamehStory.tsx` به‌روز کنید
